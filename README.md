Para testar o app em localhost:5000 basta executar o comando no repositório:
```bash
    $ python run.py
```
Ou usando o Flask environment:
(Linux/Bash):
```bash
    ~$ export FLASK_APP=run
    ~$ flask run
```
(Windows/Powershell):
```powershell
    > $env:FLASK_APP="run"
    > flask run
```
